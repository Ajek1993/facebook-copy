# Facebook-copy with AI chat

## Technologies:

- Typescript,
- Next.js,
- Tailwind CSS,
- CSS,
- Next UI,
- ChatGPT 3.5,
- ESLint,
- Firebase,
- Firestore

## Features

The application is similar to well-known social network. This app makes it possible:

- add/delete post,
- like/unlike posts,
- chat with AI assistant named John Smith,
